How do I buy Lorem?

Lorem is only available on Binance Smart Chain. To use Binance Smart Chain, you must configure your metamask or trust wallet to use the Binance Smart Chain network. Then you would need to own BNB tokens for gas fees. Finally, you would need to swap for Lorem on an exchange like PancakeSwap.

What slippage should I set on BSC?

Set Slippage 6% while trading Lorem.

What is the slippage?

The volatility in price can create such a situation where the executed price is different from the quoted and expected price. Slippage is the expected % difference between these intended and executed prices.
